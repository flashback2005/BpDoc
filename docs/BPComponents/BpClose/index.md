## 关闭图标

> 可以用作 menu 过度动画

| 属性    | 类型         | 描述       |
| ------- | ------------ | ---------- |
| opening | boolean      | 是否打开中 |
| color   | string       | 颜色       |
| size    | 'sm' \| 'md' | 尺寸       |

### 使用方法

```jsx
<BpClose :opening="true" size="md" class="" @click="doHide" />
```
